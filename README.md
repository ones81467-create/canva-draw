Canvas Sketchpad

An interactive **HTML5 Canvas drawing application** built with JavaScript.
Users can draw freely with the mouse, change brush colors in real time, and clear the canvas — demonstrating core canvas rendering and mouse event handling in a simple paint-style app. Features

* Draw on canvas using mouse
* Change brush color dynamically
* Smooth round brush strokes
* Clear the entire canvas
* Lightweight and beginner-friendly code

Technologies Used

* HTML5 Canvas
* JavaScript (Mouse Events)
* CSS
How to Run

1. Download or clone the repository
2. Open `index.html` in any modern web browser
3. Start drawing!

No installation or dependencies required.

 Concepts Demonstrated

* Canvas API (`getContext`, `lineTo`, `stroke`, `clearRect`)
* Handling mouse events (`mousedown`, `mousemove`, `mouseup`)
* Dynamic styling using input controls
* Basic drawing application logic
